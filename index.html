Primer

Primer is a simple functional programming language with dynamic typing, lexical scope, immutable bindings, lambda expressions, closures and tail calls optimization.

To get an idea of how Primer code looks, here's an implementation of Quicksort: 

def qsort = fn (l)
  if l != nil then
    def x = head(l), xs = tail(l)
    qsort(filter(fn (a) a < x end, xs))
      ++ [x]
      ++ qsort(filter(fn (a) a >= x end, xs))
  else
    []
  end
end
