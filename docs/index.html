Primer

Primer is a little programming language in the early stages of development. It's available for Linux, Mac and Windows. You can also download the source code. If you have any comments, please email me.

To get started, unpack the archive somewhere on your PATH and create the environment variable PRIMER_LIBRARY_PATH which should point to StandardLibrary.pri.

The keyword def interns a new symbol and binds it to an expression. In the example below we define a function with the lambda keyword and bind it to the symbol Double. A function implicitly returns the value of its last expression. Show is one of the special forms provided by the runtime. 

def Double = lambda (x)
    2 * x;
end

Show(Double(2))

Save your program in a file called Double.pri and at the shell enter: 

$ primer Double.pri

Primer is dynamically typed and supports integers, floats, booleans and the value nil which represents nothing. Bindings in Primer are immutable although the language is lexically scoped so they may be shadowed. 

def pi = 3.14
Show(Double(pi))

Lists can contain values of any type, in any combination and can be nested. You can construct lists literally or using the List and Cons special forms. Lists can be concatenated using the ++ operator. 

def nums = [1,2,3,4,5]

List(true)
[0,1] ++ [2,3,4] ++ [5,[6,7,8]]
Cons(Cons([0,1], [2,3,4]), [5,[6,7,8]])

You can define higher order functions.... 

def SumWithFun = lambda (fn, x, y)
    fn(x) + fn(y);
end

SumWithFun(Double, 4, 5)

...as well as anonymous functions. 

SumWithFun(lambda (x) x * 2; end, nums)

The standard library comes with a growing collection of functions:

  • Map uses a function to transform a lists values
  • Filter uses a predicate function to filter values from a list
  • Any returns true if any items from a list pass a test
  • All returns true if all items from a list pass a test
  • FoldL use a function to combine list elements
  • FoldR use a function to combine list elements
  • Sum returns the sum of all elements in the list
  • Empty returns true if the list is empty
  • Nth returns the nth element of a list
  • Find searches a list for a value
